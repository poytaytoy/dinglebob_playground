» Syntax Cheat Sheet
Variables: let x = 10;
Functions: define add(a, b) { return a + b; }
Lambdas: let f = lambda(x) { return x * x; };
Conditionals: if x > 0 { ... } else { ... }
Loops:
  • while x < 10 { ... }
  • for (let i = 0; i < n; i = i + 1) { ... }
Lists: let list = [1, 2, 3]; (Access: list[0])
Built-ins: print x;, len(l), append(l, v), concat(a, b), abs(n), timeit(expr)
