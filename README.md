
# DingleBob Playground

A web-based interactive playground for the DingleBob programming language.

## Features

- Example code buttons (sorting, struct, class, Z-combinator)
- Auto-pairing for brackets, quotes, and parentheses
- Auto-indentation on new lines
- Tab/Shift+Tab for indent/unindent
- Ctrl+Enter to run code
- Output copy button
- Code persists in localStorage
- Vertically resizable code editor

## Tech Stack
- HTML, CSS, JavaScript (Vanilla)
- Backend: Python (Flask or similar, for code execution)

## Usage
1. Write or load example DingleBob code in the editor.
2. Press **Run** or **Ctrl+Enter** to execute.
3. Copy output with the **Copy** button.
4. Code is saved automatically in your browser.

## Development Highlights
- Implemented custom code editor features (auto-pair, auto-indent, etc.) without external libraries.
- Focused on user experience and productivity for language learners.
